URL Shortener API - Endpoints

/v1/url-management/shorten

Input example


            {
                "payload": "https://github.com/topics/amiga?l=python&o=asc&s=updated"
            }

Output example


            {
              "is_url": true,
              "original": "https://github.com/topics/amiga?l=python&o=asc&s=updated",
              "shortened": "tier.app/7"
            }

/v1/url-management/route

Input example


            {
                "payload": "thisisnotanurl"
            }

Output example


            {
              "error": "Invalid input - URL not in DB.",
              "in_database": false,
              "shortened": "thisisnotanurl"
            }
